a. What is GIT? Why do we need source code Version Control System?

Git is an open source project that was developed in 2005. It is a distributed version control system designed to handle any size project. Git can help software developer keep track of all the changes in code throughout the project. A Version Control System is a system that records changes to a file or set of files (code) over the life of the project so you can get back to the old files at a later time. We need Version Control System, such as Git, the full file is mirrored in the repository. If there are many developer working on the file, we can see where something broke or can see what changes were made. Also, if one computer were to crash, we could open the repository on another computer.
