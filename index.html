What is Docker

Docker is an open-source tool that uses containers, packages parts of applications into one, to make, deploy, and run applications. One major container that Docker uses is Kubernetes. It is a major tool for DevOps. https://opensource.com/resources/what-docker

What is Nginx

Nginx is an open-source web server. It can also load balance, HTTP chaching, and be used as a reverse proxy. https://medium.freecodecamp.org/an-introduction-to-nginx-for-developers-62179b6a458f https://kinsta.com/knowledgebase/what-is-nginx/

What is VM (virtual machine)

A VM is a computer file that runs like an actual computer. The software inside a VM does not interact outside of the Vm, thus the VM is similar to a computer inside a computer. It is an ideal environment to test operating systems as it does not interact with any software outide of the VM. https://azure.microsoft.com/en-us/overview/what-is-a-virtual-machine/

Who is Cloud Computing Service Provider - Digitalocean

DigitalOcean is a cloud infrastucture based in New York. They offer an Infrastructure as a Service to software developers, where they provide virtualized computer resources over the internet. https://www.quora.com/What-is-DigitalOcean
